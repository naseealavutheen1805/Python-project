# 📊 Sales Data Analysis using Google Colab

## 📌 Project Overview
This project focuses on cleaning and analyzing a sales dataset using Python in Google Colab.  
The objective is to preprocess the data and extract meaningful business insights such as revenue trends, cancellation rates, and sales performance by location and time.

---

## 🛠 Tools & Technologies
- Python
- Pandas
- Matplotlib
- Google Colab

---

## 🔄 Data Cleaning Steps
- Converted `Date` column to datetime format (MM-DD-YY)
- Replaced missing values in `Amount` column with the median
- Checked for null values and validated data types
- Extracted Month and Weekday from Date for analysis

---

## 📈 Analysis Performed
- Total Revenue Calculation
- Total Orders Count
- Cancelled Orders & Cancellation Rate
- Monthly Revenue Analysis
- Highest Sales Weekday
- Top Cities by Revenue
- Quantity Sold by Category & Size

---

## ▶ How to Run
1. Open the notebook in Google Colab.
2. Upload the `sales_dataset.csv` file when prompted.
3. Run all cells sequentially.
4. View the output results and visualizations.

---

## 🎯 Key Insights
- Identified monthly revenue trends.
- Determined peak sales days.
- Measured cancellation rates.
- Analyzed product size and category performance.

---

## 📌 Conclusion
This project demonstrates practical data cleaning, transformation, and exploratory data analysis skills using Python.  
It highlights the ability to derive actionable business insights from raw sales data.
